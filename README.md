# Sentiment Analysis on Airline Reviews using LSTM

This project analyzes airline customer reviews and classifies them as positive or negative.

## Tools Used
- Python
- Pandas, NumPy
- TensorFlow / Keras
- LSTM

## Process
1. Data collection
2. Data preprocessing
3. Tokenization & padding
4. Model building (LSTM)
5. Training & evaluation

## Result
The model predicts sentiment from customer reviews with good accuracy.
